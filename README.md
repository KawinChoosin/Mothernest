# INTRODUCTION

## MotherNest

***

### 👋 Introduction

MotherNest คือแพลตฟอร์มดิจิทัลสำหรับคุณแม่ ผู้ปกครอง และผู้ดูแลเด็กในช่วงเติบโตแรกเริ่ม\
แพลตฟอร์มนี้รวมข้อมูลสุขภาพ การติดตามพัฒนาการ การขอคำแนะนำ และการเลือกซื้อสินค้าไว้ในที่เดียว

MotherNest ถูกออกแบบให้ใช้งานง่าย และเข้าถึงบริการสำคัญได้เร็ว\
ผู้ใช้สามารถอ่านบทความ ติดตามวัคซีน พูดคุยกับชุมชน ใช้งาน AI ผู้ช่วยส่วนตัว และปรึกษาแพทย์ผ่านระบบออนไลน์ได้จากแพลตฟอร์มเดียว

#### Key features

* **Article** สำหรับอ่านบทความความรู้เกี่ยวกับการดูแลแม่และเด็ก
* **Vaccine Tracking** สำหรับติดตามประวัติวัคซีนและกำหนดการสำคัญ
* **Marketplace** สำหรับเลือกซื้อสินค้าแม่และเด็กในแอปเดียว
* **Community** สำหรับแลกเปลี่ยนประสบการณ์กับผู้ปกครองคนอื่น
* **AI Chatbot** สำหรับถามตอบข้อมูลเบื้องต้นได้ตลอดเวลา
* **Doctor Consult** สำหรับปรึกษาแพทย์หรือผู้เชี่ยวชาญแบบออนไลน์

### 🌷 About MotherNest

MotherNest เป็นเว็บแอปที่พัฒนาเพื่อช่วยให้การดูแลแม่และเด็กเป็นเรื่องง่ายขึ้นในชีวิตประจำวัน\
แนวคิดหลักคือการรวม **content**, **health support**, **community**, และ **service access** ไว้ในระบบเดียว

จุดเด่นของแพลตฟอร์มคือการเชื่อมต่อหลายบริการเข้าด้วยกันอย่างเป็นธรรมชาติ\
ผู้ใช้ไม่จำเป็นต้องสลับหลายแอปเพื่ออ่านข้อมูลสุขภาพ ติดตามวัคซีน ขอคำแนะนำ หรือค้นหาสินค้าที่เหมาะสม

ในมุมของประสบการณ์ใช้งาน MotherNest เน้นความเรียบง่าย เป็นมิตร และเหมาะกับการใช้งานบนมือถือ\
โครงสร้างของแพลตฟอร์มจึงถูกออกแบบให้ค้นหาฟีเจอร์ได้ง่าย เข้าใจเร็ว และรองรับการใช้งานจริงของครอบครัวยุคใหม่
