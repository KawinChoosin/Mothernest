# Selection Rationale and Estimated Cost

**This page explains why each network device and cable type was selected and provides an estimated total cost.**

The estimate below is based on the listed unit prices in the device pages. It reflects hardware and cabling only.

### Selection rationale

#### Router

Cisco routers were selected because they provide the required throughput, enterprise reliability, and strong support for branch and headquarters connectivity.

#### Switch

Cisco and Meraki switches were selected because they provide sufficient switching capacity, enterprise management features, and PoE support for local access infrastructure.

#### Firewall

The firewall model was selected because it provides the required throughput for both sites while keeping hardware cost under control.

#### Access Point

The selected access point supports business Wi-Fi performance, secure SSIDs, and enough client capacity for employee and guest usage.

#### Wire

CAT6A was selected for copper runs because it supports 10 Gbps Ethernet over standard access distances. OM4 fiber was selected for high-speed uplinks and reliable backbone connectivity.
