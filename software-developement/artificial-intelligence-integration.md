# Artificial Intelligence Integration

โครงการนี้นำโมเดลปัญญาประดิษฐ์มาใช้สำหรับการแนะนำบทความที่เหมาะสมกับคุณแม่ตั้งครรภ์แบบเฉพาะบุคคลโดยประเมินจากความสนใจและต้องปลอดภัยสอดคล้องกับอายุครรภ์ทางการแพทย์

### Article Recommendation System

#### Dataset

* Item Data: ข้อมูลบทความ (เนื้อหา, หมวดหมู่, ช่วงอายุครรภ์เป้าหมาย)
* User Data: ประวัติการเข้าชมบทความของผู้ใช้งาน และอายุครรภ์ปัจจุบัน

#### Model

* ใช้สำหรับจับคู่ความสนใจของคุณแม่กับเนื้อหาบทความได้อย่างแม่นยำ
* ใช้สถาปัตยกรรมแบบ Two-Tower Neural Network
  * พัฒนาด้วยไลบรารี `PyTorch` สำหรับสร้างโครงข่ายประสาทเทียม
  * ใช้ `SentenceTransformer` (โมเดล `all-MiniLM-L6-v2`) สำหรับแปลงข้อความ (Text) เป็นตัวเลข
  * ใช้เทคนิค Shared Embedding แปลงตัวเลขอายุครรภ์เป็นเวกเตอร์ 64 มิติ เพื่อแก้ปัญหาโมเดลติดกับดักสมการเส้นตรง (Linear Trap)
* ข้อจำกัดของการใช้งาน
  * อาจเกิดภาวะ Cold Start กับผู้ใช้ใหม่ที่ยังไม่มีประวัติการอ่าน (ระบบจะสลับไปใช้กฎเกณฑ์ทางการแพทย์ หรือ Rule-based ในการแนะนำแทนชั่วคราว)
*   ตัวชี้วัดประสิทธิภาพ (Metrics)

    * ใช้ `HR@10` (ความแม่นยำ) และ `NDCG@10` (ความสามารถในการจัดอันดับ)

#### หลักการทำงานของระบบ

<figure><img src="../.gitbook/assets/image (11).png" alt=""><figcaption></figcaption></figure>

* Item Tower (ฝั่งบทความ): สกัดฟีเจอร์ของบทความ (ชื่อบทความ, หมวดหมู่, อายุครรภ์เป้าหมาย) บีบอัดเป็นเวกเตอร์ขนาด 128 มิติ แล้วบันทึกรอไว้ในฐานข้อมูล
* User Tower (ฝั่งผู้ใช้): เมื่อผู้ใช้เปิดแอปพลิเคชัน ระบบจะนำ "อายุครรภ์ปัจจุบัน" และ "ประวัติการอ่าน 5 อันดับล่าสุด" มาคำนวณเพื่อสร้างเป็นพิกัดเวกเตอร์คุณแม่ขนาด 128 มิติ

<figure><img src="../.gitbook/assets/image (12).png" alt=""><figcaption></figcaption></figure>

* Hybrid Serving:
  * AI Score: นำเวกเตอร์ของคุณแม่และบทความมาคำนวณความเหมือน (Dot Product) เพื่อหาคะแนนความรู้ใจ
  * Medical Score: นำไปคำนวณผ่านสมการเส้นโค้งระฆังคว่ำ (Gaussian) เพื่อตรวจจับระยะห่างของอายุครรภ์
  * ระบบจะนำคะแนนทั้งสองส่วนมาถ่วงน้ำหนักรวมกัน เพื่อให้ได้บทความที่ดีที่สุด ซึ่ง "ตรงใจและปลอดภัย" สำหรับคุณแม่

#### System Output

<figure><img src="../.gitbook/assets/image (13).png" alt=""><figcaption></figcaption></figure>

จากภาพตัวอย่างการทดสอบระบบกับคุณแม่ชื่อ Andrea Green (อายุครรภ์ 26 สัปดาห์) ประวัติการอ่านล่าสุดของคุณแม่เน้นไปที่กลุ่มโภชนาการ (DHA, ธาตุเหล็ก) และการเตรียมตัว (วัคซีน) ระบบ User Tower สามารถจับแพทเทิร์นความสนใจนี้ได้ จึงแนะนำบทความในหมวดหมู่ที่สอดคล้องกัน เช่น "เทคนิคจัดการแคลเซียม" และเนื้อหาที่เจาะลึกเรื่องวัคซีนกับ DHA อย่างต่อเนื่อง

แม้ AI จะหาบทความที่ตรงความสนใจได้ แต่ระบบ Hybrid Scoring (สมการเส้นโค้งระฆังคว่ำ) ได้เข้ามาควบคุมผลลัพธ์สุดท้าย จะเห็นได้ว่าบทความที่ถูกแนะนำ มีอายุครรภ์เป้าหมายอยู่ที่ 26, 29 และ 22 วีค ซึ่งเกาะกลุ่มใกล้เคียงกับอายุครรภ์ปัจจุบันของคุณแม่ (26 วีค)
